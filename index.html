Brup suite

Burp Suite is a set of tools used for penetration testing of web applications It is the most popular tool among professional web app security researchers and bug bounty hunters

OWASP ZAP

OWASP ZAP is an open-source web application security scanner. It is intended to be used by both those new to application security as well as professional penetration testers

Snort

Snort is the foremost Open Source Intrusion Prevention System (IPS) in the world. Snort IPS uses a series of rules that help define malicious network activity

Nessus
